# CineTech
CineTech: C# WinForms ve MSSQL ile geliştirilmiş, N-Tier mimarili kapsamlı sinema otomasyonu. Yönetici, Personel ve Müşteri panelleri bulunur. Görsel koltuk seçimi (Boş/Dolu/Bakım), akıllı seans filtreleme, biletleme ve detaylı raporlama özelliklerine sahiptir. Veritabanı yönetimi ve OOP prensipleriyle tasarlanmıştır.
